# GSローカル環境構築手順書

## 環境
- Mac or Windows
- DockerDeskTop
- 

## 構築手順

1. ソースを取り込む

2. myappディレクトリに移動し、
`docker-compose up --build`
を実行する

3. [http://localhost:3000/](http://localhost:3000/)にアクセスする
※終了する際は `control+c` を入力してください
